What is Kali Chroot

Kali Chroot: Chroot is essentially a full Kali Linux installation that allows Android to share network, storage, and other resources.
To manage Chroot we usually use Kali Chroot Manager.

The NetHunter chroot manager allows you to download and install a Kali Linux chroot (if one does not already exist), backup and restore a chroot, as well as remove an existing chroot. In addition, one can install various Kali Linux metapackages as required.

Requirements

Requirements
1.Mind Presence
2.Rooted Mobile Device

Setup

Now the main process starts..

So firstly go to your Nethunter Chroot Manager and Stop Kali Chainroot. [If you have installed it already]
Now you need to install the Apps2SD Pro apk in your mobile device.
Download link provided in the requirements section above.

After Installation open the app and go to Partition Tool and part your SD Card in 2 partitions..
FILE SYSTEM of 1st partition will be FAT32 and FILE SYSTEM of 2nd partition will be EXT4.


The 1st partition will be use for storing media and other stuff and the 2nd partition will be use for storing your Nethunter Files and Other Apps.
If you use Full Kali FS then make 2nd partition bigger eg. 30-35GB
After doing partition, go to Link Apps to SD Card and create mount script for your second partition..
You don't need to do anything the app will prompt you on it's own. Just choose the options according to you.

After creating Mount script, get back to main menu and then go to Folder Mount Option and Tap on the + icon in it.
After that an interface like the image will be in front of you, then fill it like the image...

Name: Nethunter
Source Folder: /data/local/nhsystem
Destination Folder: /data/sdext4/.Nethunter
Adding "." in front of folder Name will prevent it from Media Scan
Also Select the OPTIONS same as Image attached above!

Save the mount file by pressing "OK", Now toggle the ON/OFF button and select Move Data and wait until it's done.
Note: To make this process fast don't install Chroot at first. Flash is after you setup folder mount and enable it.

BOOM! Now your Chroot is in SD card just go to Kali Chroot Manager and Start it and ENJOY HACKING.

In case your Nethunter app show CHROOT not Installed after reboot or something.
Go to Apps2SD and then Folder Mount and press on 3 Dots and tap on Mount ONLY(ALL)
And do a soft reboot and then go to Kali Chroot Manager and start the Chainroot.
